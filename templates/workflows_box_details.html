Box Folder Workflow Guide

Today: {{ today }} Version: {{ app_version or 'n/a' }}

Workflow Overview

  1. Create folder via POST /box/folder/create using the deal and contact names.
  2. Tag metadata with POST /box/folder/tag once you have the new folder_id.
  3. Share Client Sharing subfolder through POST /box/folder/share (optional until automation is enabled).

Each HubSpot custom-code action receives the same payload shape. You can reuse the dictionary of HubSpot fields across all three calls, and be sure to include the spouse identifiers, names, and email every time.

Endpoint Input Reference

Endpoint Minimum fields Notes
/box/folder/create hs_deal_record_id, primary first/last, spouse first/last, deal_salutation All names provided in payload; no HubSpot lookup needed.
/box/folder/tag folder_id, contact & spouse IDs, names, emails, deal_salutation, household_type Payload supplies the entire metadata template; no HubSpot API call.
/box/folder/share folder_id, hs_contact_email, hs_spouse_email Endpoint invites both primary and spouse emails to Client Sharing.

HubSpot Custom Code Actions

Set up three custom-code actions in your workflow—one for each Box endpoint. Reuse the field lists shown below and pass the folder_id output from Action 1 into Actions 2 and 3.

Action 1 – POST /box/folder/create

hs_deal_record_id hs_contact_firstname hs_contact_lastname hs_spouse_firstname hs_spouse_lastname deal_salutation

Only the fields shown here are required to create the folder without HubSpot lookups.

import requests

BOX_BASE = "https://<your-app-domain>"

def main(event):
    fields = {**(event.get("fields") or {}), **(event.get("inputFields") or {})}
    deal_id = event.get("deal_id") or fields["hs_deal_record_id"]

    payload = {
        "deal_id": str(deal_id),
        "fields": {
            "hs_deal_record_id": str(deal_id),
            **{key: str(fields[key]) for key in (
                "hs_contact_firstname", "hs_contact_lastname",
                "hs_spouse_firstname", "hs_spouse_lastname",
                "deal_salutation",
            )},
        },
    }

    resp = requests.post(
        f"{BOX_BASE}/box/folder/create",
        json=payload,
        timeout=20,
    )
    resp.raise_for_status()
    folder = resp.json().get("folder", {})

    return {
        "outputFields": {
            "folder_id": folder.get("id"),
            "folder_url": folder.get("url"),
            "folder_name": folder.get("name"),
        }
    }

Action 2 – POST /box/folder/tag

hs_deal_record_id folder_id hs_contact_id hs_contact_email hs_contact_firstname hs_contact_lastname hs_spouse_id hs_spouse_email hs_spouse_firstname hs_spouse_lastname deal_salutation household_type

`folder_id` comes from Action 1; include the other fields to keep metadata tagging self-contained.

import requests

BOX_BASE = "https://<your-app-domain>"

def main(event):
    fields = {**(event.get("fields") or {}), **(event.get("inputFields") or {})}
    deal_id = event.get("deal_id") or fields["hs_deal_record_id"]
    folder_id = event.get("folder_id") or fields["folder_id"]

    payload = {
        "deal_id": str(deal_id),
        "folder_id": str(folder_id),
        "fields": {
            "hs_deal_record_id": str(deal_id),
            **{key: str(fields[key]) for key in (
                "hs_contact_id", "hs_contact_email",
                "hs_contact_firstname", "hs_contact_lastname",
                "hs_spouse_id", "hs_spouse_email",
                "hs_spouse_firstname", "hs_spouse_lastname",
                "deal_salutation", "household_type",
            )},
        },
    }

    resp = requests.post(
        f"{BOX_BASE}/box/folder/tag",
        json=payload,
        timeout=20,
    )
    resp.raise_for_status()

    return {
        "outputFields": {
            "folder_id": str(folder_id),
            "metadata_status": "applied",
        }
    }

Action 3 – POST /box/folder/share

folder_id hs_contact_email hs_spouse_email

Provide both contact and spouse emails so the Client Sharing subfolder can be shared without additional lookups.

import requests

BOX_BASE = "https://<your-app-domain>"

def main(event):
    fields = {**(event.get("fields") or {}), **(event.get("inputFields") or {})}
    folder_id = event.get("folder_id") or fields["folder_id"]
    emails = [
        fields[name]
        for name in ("hs_contact_email", "hs_spouse_email")
        if fields.get(name)
    ]

    resp = requests.post(
        f"{BOX_BASE}/box/folder/share",
        json={"folder_id": str(folder_id), "emails": emails},
        timeout=20,
    )
    resp.raise_for_status()

    return {
        "outputFields": {
            "folder_id": str(folder_id),
            "shared_emails": emails,
        }
    }

Required HubSpot Properties

  • hs_deal_record_id
  • hs_contact_id, hs_contact_email, first/last name
  • Spouse fields (hs_spouse_id, hs_spouse_email, first/last name)
  • deal_salutation, household_type
